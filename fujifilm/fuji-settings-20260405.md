Fuji settings 2026-04-05

How to reload:
- Use Fujifilm Tether https://www.fujifilm-x.com/en-us/support/download/software/tether-app/
- Connection setting > Connection mode = USB raw conv/backup restore

Notable things changed
- Sound setting
  - AF beep volume: off
  - shutter volume: off
- Back button focus on AF-C only https://www.kevinmullinsphotography.co.uk/blog/fujifilm-back-button-focusing-guide
- Button/dial setting
  - Function (fn) setting
    - Fn2: playback
    - Fn6: wireless communication
    - AF-L: AF-ON (for back button focus)
  - Command dial setting
    - front one: aperture (F)
- Highlight alerts https://www.jmpeltier.com/fujifilm-live-view-highlight-alert/
  - Screen setting > Disp. custom setting > Live view highlight alert = on
  - Screen setting -> Natural live view = on (jpeg preview)
